# Perfil do desenvolvedor — Agile360

## Estilo de trabalho
- Prefere **resultados diretos** — "Não me pergunte nada. Construa. Surpreenda."
- Aprecia **audácia criativa** no design — quer que o produto se destaque no mercado
- Dá briefings completos e detalhados; espera execução fiel + surpresas positivas
- Não quer perguntas desnecessárias durante a implementação

## Nível técnico
- Desenvolvedor com conhecimento sólido de React, TypeScript, CSS
- Entende design system, tokens CSS, arquitetura de componentes
- Usa AIOX (Synkra) para orquestração — familiarizado com o framework

## Preferências de design
- **Minimalismo sofisticado** — menos é mais, mas cada detalhe deve ter presença
- **Referências:** Linear.app, Vercel, Stripe (nível de polish)
- **Tipografia dramática** — Instrument Serif italic para momentos de impacto
- **Paleta Ouro Jurídico** — comprometido com #C9A84C como identidade do produto
- Detesta "AI slop" — genérico, Inter/Roboto em tudo, gradientes roxos
